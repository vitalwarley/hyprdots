## Session Documentation Process

When creating diary entries with the `claude-memory:diary` skill:

1. **Create the diary entry** using the skill (saves to `~/.claude/memory/diary/`)
2. **Copy to project**: Copy the diary to `.sessions/claude/<datetime>-<title>.md`
3. **Update SESSIONS.md**: Add an entry with date, file path, and summary
4. **Verify CLAUDE.md**: Ensure this file references SESSIONS.md (already done above)
